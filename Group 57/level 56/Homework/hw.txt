ისწავლეთ ახალი სიტყვები:

call - გამოძახება
პარამეტრი - გადავცემთ ფუნქციას შექმნისას
არგუმენტი - გადავცემთ ფუნქციას გამოძახებისას
function - ფუნქცია - კოდი, რომელიც ასრულებს რაღაც კონკრეტულ საქმეს
call - გამოძახება - ფუნქციის დაძახება, რომ იმუშაოს
parameter - პარამეტრი - ცვლადი, რომელიც განსაზღვრულია ფუნქციაში შექმნისას
argument - არგუმენტი - მონაცემი, რომელსაც გადავცემთ ფუნქციას გამოძახებისას
return - დაბრუნება - ფუნქციის მიერ მიღებული შედეგის უკან დაბრუნება
scope - მოქმედების არეალი - ადგილი, სადაც ცვლადი "ჩანს" და იძლევა წვდომას
default value - ნაგულისხმევი მნიშვნელობა - მნიშვნელობა, რომელსაც პარამეტრი მიიღებს, თუ არგუმენტი არ გავუგზავნეთ
recursion - რეკურსია - როცა ფუნქცია თავის თავს იძახებს
anonymous - ანონიმური (უსახელო) ფუნქცია - ფუნქცია რომელსაც არ აქვს სახელი

variable - ცვლადი - კონტეინერი, რომელიც ინახავს მონაცემს
value - მნიშვნელობა	ის მონაცემი, - რაც ცვლადშია შენახული
constant - მუდმივი (ცვლადი) - ცვლადი, რომლის შეცვლაც აღარ შეიძლება
data type - მონაცემის ტიპი - მონაცემის სახეობა (მაგ. ტექსტი, რიცხვი)
string - სტრინგი (ტექსტი) - ტექსტური მონაცემი
number - რიცხვი - ციფროვანი მონაცემი
boolean	ბულიანი (true/false) - ჭეშმარიტი ან მცდარი
array - მასივი - ელემენტების სია
object - ობიექტი - მონაცემთა კოლექცია გასაღები-მნიშვნელობით

loop - ციკლი - კოდი, რომელიც რამდენჯერმე სრულდება
condition - პირობა - მართავს, რომელი კოდი შესრულდეს
if - თუ	- პირობის შემოწმება
else - წინააღმდეგ შემთხვევაში - როცა პირობა არ შესრულდება
event - მოვლენა	- მაგ. როცა დავაჭერთ ღილაკს
input - შეყვანა	რაც მომხმარებელი ან კოდი აწვდის
output - გამოტანა - შედეგის ჩვენება ან დაბრუნება
debug - შეცდომების პოვნა - კოდის გადამოწმება და გასწორება
